When opened the programme displays the main form.


Figure 1. The Main Form

Adding A Contact

To add a new contact click the Add button pon the toolbar or select Contacts|Add from the menu. This clears the display and changes the toolbar to show Ok and Cancel buttons.

Add the contact detail and then click the Ok button on the toolbar. This will add the contact to the addressbook database. Clicking the cancel button will cancel the add operation.

Editing a Contact

To edit a contact ensure the contact you wish to edit is displayed and then click the Edit button on the toolbar or select Contacts|Edit from the menu. All data is left in place and changes the toolbar to show Ok and Cancel buttons

Deleting A Contact

To delete a contact ensure the contact you wish to deleteis displayed and then click the Delete toolbar button or select Contacts|Delete from the menu. The contact will be permanently deleted.

Searching For A Contact

To search for a contact click the search toolbar button or select Contacts|Search from the menu. This will display the search criteria form.


Figure 2. search Criteria

Enter a full or partial first and\or lastname and or select a country from the drop down box and then click the Ok button. All contacts that match the criteria will then be displayed on the main form one by one.

Page Setup

The margins to be used when printing the address book are setup by selecting Contacts|Page Setup from the menu. This allows you to specify top,bottom,left and right margins for the printed address book.

Print Preview.

Select the print Preview button on the toolbar or select print preview from the menu. This will display a prinrer requester where you can select the target printer and paper size and orientation. Once selected the address book will be shown in a print preview window. Only the currently selected contacts are shown as selected by any search operation conducted previously. The preview includes the thumbnails for each contact as well.

Print

Printing the address book is similar to the above but select Contacts|Print from the menu. No preview is shown and the currently selected contacts are sent to the printer.

Help

Click the Help toolbar button to display this help file.

Exit

Click the Exit toolbar button to exit the programme.
